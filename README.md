# Thymer-Catppuccin-Latte-Frappe-Macchiato
These themes are designed for the Thymer notes application.
Designed around the [Catppuccin colour palette](https://github.com/catppuccin/palette), these themes are inspired by the [Catppuccin Mocha Theme](https://github.com/niklashog/Thymer-Catppuccin-Mocha) for Thymer by jd_thymer and [Niklas](https://github.com/niklashog).

## Contents
  - Thymer Catppuccin Latte CSS - Catppuccin's light theme
  - Thymer Catppuccin Frappé CSS - Catppuccin's softer dark theme
  - Thymer Catppuccin Macchiato CSS - Catppuccin's standard dark theme
  - Thymer Catppuccin Combined CSS - All three combined for easier copy/pasting

## How to apply:

1. Open the desired theme CSS file in this repository and copy (Ctrl + C) all of its contents.
3. Open up Thymer. Press Ctrl + P > Edit Theme CSS > paste (Ctrl + V) > Click Save.
4. Press Ctrl + P again > Set Theme > Catppuccin Latte.

## Comparrison:
All three of these themes are considerably lighter than Mocha's extra dark palette. Here is a quick overview if you are unfamiliar.

### Latte - Catppuccin's light theme
- Soft, creamy backgrounds instead of harsh white
- Dark text with gentle contrast
- Designed to feel warm, calm, and paper-like
- Easier on the eyes than many light themes
