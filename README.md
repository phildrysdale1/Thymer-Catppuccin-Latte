# Thymer-Catppuccin-Latte
Catppuccin Latte Theme for Thymer notes application. The IDE for Notes, Tasks and Planning.

This is a Thymer theme inspired by the [Catppuccin Mocha Theme](https://github.com/niklashog/Thymer-Catppuccin-Mocha) for Thymer by jd_thymer and [Niklas](https://github.com/niklashog).

## How to apply:

1. Open the Thymer Catppuccin Latte CSS in this repository and copy (Ctrl + C) all of its contents.
2. Open up Thymer. Press Ctrl + P > Edit Theme CSS > paste (Ctrl + V) > Click Save.
3. Press Ctrl + P again > Set Theme > Catppuccin Latte.
